# 變更紀錄 Changelog

本檔記錄「國際交流成效回報系統」每一版的變更。
格式參考 [Keep a Changelog](https://keepachangelog.com/)，版本號採語意化版本（[SemVer](https://semver.org/)）：`主版號.次版號.修訂號`。

- **主版號（major）**：不相容的重大改版、資料結構或流程的破壞性變更。
- **次版號（minor）**：新增功能或欄位，且向下相容。
- **修訂號（patch）**：修錯字、修 bug、樣式微調。

> 版本號不需手動維護：每次改動程式碼時，由做變更的 AI（Claude Code，或對話中的 Claude）依本次實際變更判斷 SemVer 等級，更新 `lib/version.ts` 的 `APP_VERSION`，並在下方補上一段該版紀錄。詳細規則寫在 `CLAUDE.md`。

## [1.1.0] - 2026-06-17
### 新增 Added
- 後台新增「選項管理」分頁：可增刪／停用「學制、活動大類、活動類型」項目；活動大類可設定「指標歸組」（出國交流人次 vs 研討會場次/人數），活動類型連動所屬大類。
- 填報「國家/地區」改為可搜尋下拉：固定清單來自教育部國別代碼（203 國家/地區），可用中文／英文／代碼搜尋，亦允許自由輸入。
- 學院頁與後台總覽顯示「填報中」狀態（已有資料但尚未送出），並每 30 秒自動更新、可手動「立即更新」，不必等各系送出即可看到目前進度；填報頁亦每 30 秒同步全系共用清單。
### Changed
- 學制／活動大類／活動類型改由資料庫 `options` 表維護（取代硬編碼常數），API 端依此驗證；三指標彙總視圖改用大類的 `metric_group` 分組以支援自訂大類。
- 移除 `activities.degree` / `activities.category` 的固定 CHECK 限制（改由選項表與 API 驗證）。
### 升級需執行 Migration
- 於 Supabase SQL Editor 執行 `db/migrations/2026-06-17_options.sql`（新建 options 表、重建彙總視圖、移除舊 CHECK、寫入預設選項）。

## [1.0.0] - 2026-06-17
### 新增 Added
- 國際交流成效回報系統首次正式版。
- 系所端免登入填報：全系共用一份清單，任何人都能整理未送出內容，確認送出即整批鎖定。
- 每筆紀錄記錄填報人；填報頁可即時看到全系所有項目與三指標小計。
- 學院彙整檢視、國際處後台儀表板、Excel 匯出。
- 入口首頁底部頁尾：開發者／維護資訊、版權說明、版本號與最後更新日期。
